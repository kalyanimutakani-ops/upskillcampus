# IoT-Based Temperature and Humidity Monitoring System

## Description
This is a beginner Embedded Systems and IoT project using a DHT11 sensor with an ESP32 (or ESP8266). The system reads temperature and humidity and displays the values through the serial monitor. It can be extended to send data to cloud platforms.

## Components
- ESP32 / ESP8266
- DHT11 Sensor
- USB Cable
- Jumper Wires

## Software
- Arduino IDE
- DHT Sensor Library

## Working
1. Read temperature and humidity from the DHT11 sensor.
2. Display readings on the Serial Monitor.
3. (Optional) Upload readings to an IoT cloud service.

## Future Improvements
- Connect to ThingSpeak or Blynk.
- Add LCD/OLED display.
- Send alerts when temperature exceeds a threshold.
